## [Addictol](https://www.nexusmods.com/fallout4/mods/84214?tab=description)
:material-tag-outline: `Required`

Fixes numerous engine bugs and scripts and provides helpful crash logs when you game crashes.

### Installation Instructions
* Download and install **[Main Files - Addictol :material-tray-arrow-down:](https://www.nexusmods.com/fallout4/mods/84214?tab=files)** with your mod manager
    * Select "OG" during the installation
* Download and install **[Main Files - Addictol Crash Logger :material-tray-arrow-down:](https://www.nexusmods.com/fallout4/mods/84214?tab=files)** with your mod manager

???+ warning "Did you install Microsoft VC++?"
    Without the Microsoft VC++ (x86 and x64 versions) which was mentioned in the requirements section of the guide, Addictol and the xSE Preloader will not work properly. Make sure that you've installed it!
